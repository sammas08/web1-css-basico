¿Que significa CSS?

-Es un lenguaje que sirve para describir la presentacion y el estilo de los documento HTML.

¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS sirve para dar los estilos (diseño) necesarios para que un sitio web o una aplicación web se vean visualmente bien.

El programa que se utiliza para interpretar el CSS es HTML.

¿A qué se llama regla de CSS?

Las reglas son un conjunto de propiedades y valores que se aplicarán a una o varias etiquetas según necesitemos.

¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos

La regla está compuesta por los siguiente elementos:

  1. Selector: selecciona los elementos a dar estilo (en este caso, los elementos p ). Para dar estilo a un elemento diferente, solo cambia el selector.
  2. Declaración: aplica las propiedades que querés dar al elemento seleccionado en el Selector.
  3. Propiedades: es lo que te permite dar un estilo a un elemento HTML.
  4. Valor de la propiedad: las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

¿Qué es un selector en CSS?

un selector es aquel que nos permitirá aplicar estilos a elementos específicos de HTML

Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector ID: el atributo ID de HTML identifica a un elemento HTML por medio de un nombre.

Selector de clase: te permite seleccionar y aplicar estilos a todos los elementos de HTML que comparten un atributo class especifico.

Selector universal: Nos permite aplicar estilos a todos los elementos HTML de un documento.

Selector de etiquetas: permite seleccionar todos los elementos HTML de un tipo especifico.

Selector descendente: permite seleccionar elementos que son descendientes de otro elemento especifico.

Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

El CSS lee el HTML, encuentra las reglas CSS, las aplica al HTML segun la cascada y luego muestra la pagina con el estilo correcto.

  1. El navegador carga el HTML
  2. Convierte el HTML en un DOM
  3. el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los videos incrustados.
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página

Utilizaremos las siguientes propiedades de CSS. Completar la columna de valores posibles para cada propiedad.

propiedad descripcion valores posibles
color Define el color del texto. color: rgb(34, 12, 64, 0.6), color: hsl(30, 100%, 50%, 0.6), color: red
background-color Define el color de fondo de un elemento color: rgb(34, 12, 64, 0.6), color: hsl(30, 100%, 50%, 0.6), color: red
font-size Define el tamaño de la fuente del texto. xx-small, larger, porcentaje
font-weight Define el grosor de la fuente del texto (por ejemplo, si se trata de texto en negrita o normal). font-weight: lighter;,font-weight: bold;, font-weight: normal;
font-style Define el estilo de la fuente del texto (por ejemplo, si es cursiva o normal). bolder, .oblique, .normal
font-family Define el tipo de letra a utilizar para el texto. Georgia, Courier, cursive
text-align Define la alineación horizontal del texto (izquierda, centro o derecha) left, center, end
text-decoration Define la decoración del texto (por ejemplo, si está subrayado o tachado). underline red;, none;, underline wavy red;
text-transform Define la transformación del texto (por ejemplo, en mayúsculas o minúsculas). capitalize;, lowercase;, full-width;
letter-spacing Define el espacio entre letras. rem, em, px
line-height Define la altura de línea del texto. rem, em, px
